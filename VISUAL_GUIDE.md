## 🎬 Полный сценарий использования

```
1. Пользователь открывает бота
   └─> /start

2. Видит главное меню
   └─> 3 кнопки: Рецепты, Калории, Помощь

3. Выбирает "Найти рецепт"
   └─> Бот просит ввести запрос

4. Вводит "pasta"
   └─> Бот ищет рецепты (3 сек)

5. Получает 5 рецептов
   └─> Читает подробности

6. Нажимает "Назад"
   └─> Возврат в главное меню

7. Выбирает "Подсчитать калории"
   └─> Бот предлагает способ ввода

8. Выбирает "Ввести текстом"
   └─> Бот просит описать блюдо

9. Вводит "200g chicken, 100g rice"
   └─> Бот считает калории (2 сек)

10. Получает детальный расчет БЖУ
    └─> Видит: 460 ккал, 64.7г белка

11. Доволен результатом
    └─> Использует информацию для питания

12. Нажимает "Назад"
    └─> Возврат в главное меню

13. Готов использовать бота дальше!
```
